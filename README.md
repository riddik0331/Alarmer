# ⏰ Alarmer — десктопный будильник для Windows

Красивый, удобный и надёжный десктопный будильник с тёмной темой, системным треем и поддержкой собственных звуков.

## ✨ Возможности

- **Неограниченное количество будильников** — создавай сколько нужно
- **Гибкое расписание** — единоразово, по дням недели или ежедневно
- **Snooze** — отложить на 5/10/15 минут
- **Свои звуки** — используй встроенные или загружай `.wav` / `.mp3`
- **Постепенное увеличение громкости** — плавный fade-in за 30 секунд
- **Системный трей** — сворачивается в трей при закрытии окна
- **Тёмная тема** — стильный Material Dark дизайн с анимациями
- **Автосохранение** — все будильники сохраняются между запусками

## 🚀 Быстрый старт

### Требования
- Python 3.11+
- PySide6

### Установка и запуск

```bash
# Установить зависимости
pip install PySide6

# Запустить
python src/main.py
```

### Запуск без консоли (в фоне)
Просто кликни дважды по `Alarmer_v1.bat` или `Budilnik.bat`.

### Готовый .exe
В папке `dist/` лежит собранный `Budilnik.exe` — можно запускать без Python.

## 📁 Структура проекта

```
Alarmer_v1/
├── src/
│   ├── main.py                # Точка входа
│   ├── app.py                 # Оркестратор
│   ├── models/                # Модели (Alarm, AlarmManager, SoundManager)
│   ├── views/                 # Представления (окна, карточки, попапы)
│   ├── controllers/           # Контроллеры (связь модели и вида)
│   └── resources/             # Ресурсы (иконки, звуки, стили)
├── Alarmer_v1.bat             # Запуск в фоне
├── requirements.txt           # Зависимости
└── README.md                  # Этот файл
```

## 🧪 Тестирование

```bash
pip install pytest pytest-qt pytest-cov
set QT_QPA_PLATFORM=offscreen
python -m pytest tests/ -v --cov=src.models --cov=src.utils
```

## 🛠️ Сборка .exe

```bash
pip install pyinstaller
pyinstaller Budilnik.spec
# Итог: dist/Budilnik.exe
```
